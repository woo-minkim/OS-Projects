# Phase 3 정리 문서

## 개요

`phase 3`는 File System 확장 단계입니다.  
압축본(`os_prj3_20212019.tar.gz`)을 해제해 `src`를 확보했고, 대표 보고서를 `document.docx`로 통일했습니다.

## 코드 확인 포인트

- `src/filesys`: 파일/디렉터리/inode 관련 핵심
- `src/userprog/syscall.c`: 파일시스템 연계 시스템콜 처리
- `src/tests/filesys`: 파일시스템 테스트 케이스
