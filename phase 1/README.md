# Phase 1 정리 문서

## 개요

`phase 1`은 Pintos Threads 기반 단계입니다.  
코드는 `src`에, 대표 보고서는 `document.docx`에 두고 나머지 참고 자료는 `addition`으로 분리했습니다.

## 코드 확인 포인트

- `src/threads`: 스케줄러/동기화 핵심
- `src/devices/timer.c`: 알람 관련 타이머 동작 확인 지점
- `src/tests/threads`: Phase 1 테스트 케이스
