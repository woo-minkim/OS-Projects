# Phase 4 정리 문서

## 개요

`phase 4`는 Virtual Memory 확장 단계입니다.  
압축본(`os_prj4_20212019.tar.gz`)에서 `src`를 추출해 루트 구조를 통일하고 보고서 파일명을 `document.docx`로 맞췄습니다.

- `src/tests/vm`: 가상메모리 테스트 케이스
